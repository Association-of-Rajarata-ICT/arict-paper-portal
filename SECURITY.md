# Security Policy

## Reporting a Vulnerability

If you discover a security issue, please report it privately and do not open a public issue.

Contact:

- security@arict-portal.example

Please include:

- A clear description of the issue
- Steps to reproduce
- Any relevant logs or screenshots

We will acknowledge reports within 3 business days and provide a status update once the issue is confirmed.

## Supported Versions

The latest version on the default branch is supported for security fixes.
